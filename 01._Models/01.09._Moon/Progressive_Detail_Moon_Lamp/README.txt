# Summary

This is the highest detail Moon Lamp you can find on the web. All STL is generated from 23k Moon bump map + Moon Color map.

This Moon model increase details (number of faces) progressively base on the print size. Theoretically I can generate a 24 inches Moon lamp model, but Thingiverse size limit only allow me upload the model with refinement level up to 360. So all the model over 3.5 inches are limited to refinement level 360. I think it is good enough in most case.

Please find more details at instructables:
https://www.instructables.com/id/Progressive-Detail-Moon-Lamp/
